# OpenFin Library

Angular library providing @openfin/core-web integration for the Rates E-Trading Desktop.

Provides:
- **OpenFinService** - Manages the connection to the OpenFin Web Broker, Interop context, and Layout system.
- **OpenFin configuration interfaces** - Typed configuration for broker URLs, provider IDs, and layout settings.
- **Interop helpers** - Context sharing and intent handling via the FDC3/Interop APIs.
